# docs/reports — agent digests (human-reviewed)

Digests written by the marketing skills. Committed as a running record — review before relying on
any suggested action.

- `awin-<date>.md` — weekly AWIN performance + merchant-join status (**awin-digest**, launchd
  **Mon 08:30**). A `## NEEDS-HUMAN` ACTIVATE snippet appears only when a higher-EPC merchant
  (e.g. Heathrow Airport Parking) joins — it's a *suggestion to apply by hand*, never auto-applied.
- `ai-citations-<month>.md` — whether ChatGPT / Perplexity / Gemini cite ParkMath
  (**ai-citation-check**, monthly, **interactive** via the Chrome extension — not headless).

Nothing in these reports auto-publishes or edits affiliate ranking.
