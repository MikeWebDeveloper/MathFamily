## C. AVOID (verified junk / dead / paid / spam)

- **FreeWebSubmission.com** — search-engine ping service, creates NO backlink page.
- **"300+/500+ free directory" listicle farms** (theguidex, traffictail, serpmaestro, etc.) and the
  color-named "linker" reciprocal networks (red/grey/pink/yellow-linker) — SEO spam, negative signal.
- **Dead / broken TLS:** HotVsNot, ABC Mundi, A1 Web Directory, traveldirectoryinfo.co.uk.
- **Paid-only:** Jasmine Directory ($79), DirJournal, Futurepedia (AI-only), Tap4.ai (AI+paid).
- **itravelnet** — reciprocal-link travel farm, low value.

## D. Not a form route — handle as PR/outreach instead

submit form** — inclusion is by email pitch. See `mike-guide` PART 2 (Digital PR).
